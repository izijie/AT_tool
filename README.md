一个AT命令助手软件，使用QT5开发，C++11开发，软件我将一直维护

## **使用方式**

软件代码开源，我只编译了Windows（x64）平台的可执行文件，下载AT_tool(x64).zip并解压，点开AT_tool.exe,就可以直接使用。
